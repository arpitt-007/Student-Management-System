2. index2.php,add.php,search.php,delete.php and update.php uses database studentdb where the students details are stored and profile pic is saved in folder named profile picture.

Steps to run it in your local Machine.

Step 1 : Download all the files in the repository and save it in a folder.
Step 2 : There are two databases "studentdb.sql" and "logindb.sql" . Import these databases in MySQL from phpMyAdmin.
Step 3 : All done now go to your browser and type : localhost/(Folder_name you have used) and hit enter. the website should be ready to go

Note: The folder containing all the files should be kept in path xampp/htdocs/(Folder Here)
